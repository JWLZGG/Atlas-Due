# Atlas Due - Frozen Scope

## One-line pitch
Atlas Due is a Solana-native trust and diligence layer for stablecoin settlement, treasury, and payment counterparties.

## Core user flow
A user enters a Solana wallet address, receives a structured risk summary, key flags, a score, and a one-page review memo, then can anchor a review record on Solana Devnet.

## Target customer
Platforms, treasury teams, payment operators, and stablecoin settlement workflows reviewing wallets before onboarding, routing, payout, or treasury use.

## Core problem
Stablecoin and treasury workflows increasingly rely on wallet-level trust decisions, but those reviews are still fragmented across explorers, screenshots, and internal notes.

## Key insight
Atlas Due is not just a wallet analytics tool. It transforms wallet review into a portable diligence artifact for Solana-native financial operations.

## v1 includes
- Solana wallet input
- analysis dashboard
- 4–6 deterministic risk flags
- review score
- one-page memo
- Solana Devnet attestation

## v1 excludes
- multichain support
- merchant checkout flow
- Visa/card integrations
- full compliance suite
- team accounts
- alerts
- portfolio management
- broad AI copilot workflow

## Four core screens
1. Search
2. Analysis
3. Memo
4. Attestation

## Primary demo use case
A platform or treasury team reviews a Solana settlement wallet before approving it for larger stablecoin flows or treasury routing.
